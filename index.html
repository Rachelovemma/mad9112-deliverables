Git Commands you need to know

  • git init
    Initializes a new Git repository in your current directory.
  • git status
    Displays the state of the working directory and staging area.
  • git add -A
    Prepares all changes in your working directory to be committed.
  • git commit -m ""
    Records your changes in the repository with a message.
  • git log
    Lists all previous commits with details like author and date.
  • git remote add origin <GitHub repo url>
    Links your local repo to a remote GitHub repository.
  • git push
    Sends your committed changes to the remote repository's main branch.
  • git pull
    Fetches changes from the remote repository and merges them into your current branch. This keeps your local branch up to date with the remote.
  • git clone
    Creates a local copy of a remote repository. Use this when you want to download a project from GitHub.
  • git branch
    Lists all local branches in your repository. You can also use this command to create or delete branches.
